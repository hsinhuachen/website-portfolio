# kevin websites

### 一頁式網頁

#### 優點
* 主題明確
* 動態活潑生動

#### 缺點
* 影響到SEO的排名
* 內容缺乏深度


製作時間：2 ~ 5工作天


#### 一頁式網站案例
* [衛武營國家藝術文化中心](http://project.oniondesign.com.tw/weiwuying/) 
* [牧師樓古蹟庭園咖啡](http://www.boksulao.com/)
* [愛愛搖滾音樂祭](http://project.oniondesign.com.tw/love/)


### CMS網站 (wordpress、joomla)

#### 優點
* 現有版型模組
* 維護方便

#### 缺點
* loading 速度較慢
* 轉移成本高, 不容易轉移其他系統
